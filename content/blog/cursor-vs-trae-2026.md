## 先说结论

**日常快速开发选 Trae（免费、中文好），复杂项目重构选 Cursor（生态成熟、Agent 更强）。**

---

## 价格

| | Cursor | Trae |
|---|---|---|
| 免费版 | 2000 次/月 | 完全免费 |
| Pro | $20/月 | 无付费计划 |
| 内置模型 | 自选 API Key | 内置 Claude 4 + GPT-5 |

Trae 目前完全免费，内置模型不限次数。Cursor Pro $20/月才能舒适用。

---

## 模型能力

Cursor 需要自己绑 API Key，灵活性高但也要自己花钱。Trae 内置 Claude 4 和 GPT-5，字节买单，零成本用顶级模型。

---

## 中文体验

Trae 完胜。中文注释、中文对话、中文 UI，字节团队对中文开发者体验下了功夫。Cursor 的中文支持只能说"能用"。

---

## Agent 模式

Cursor 的 Agent 模式（Composer + YOLO mode）是目前最强的，能自动读文件、执行命令、修复错误。Trae 的 Builder 模式类似但尚不如 Cursor 成熟。

---

## 适合谁

**选 Trae：** 预算有限的独立开发者、中文项目为主、快速原型开发。目前只有 macOS 版，Windows 用户需等待。

**选 Cursor：** 已有 API Key、复杂项目重构、需要 Agent 自动化、需要灵活的模型选择。

---

## 我们的建议

如果你还没付费，先用 Trae。免费、模型好、中文强。等 Trae 不够用了（特别是 Agent 能力），再考虑 Cursor Pro。
