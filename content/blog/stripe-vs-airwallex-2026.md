## 一句话总结

**有海外公司/身份 → Stripe；只有国内主体 → 空中云汇或 Paddle。**

---

## 开户门槛

| | Stripe | Airwallex | Paddle | LemonSqueezy |
|---|---|---|---|---|
| 需要海外主体 | 是（部分区域例外） | 否 | 否 | 否 |
| 支持中国大陆主体 | 否 | 是 | 是 | 是 |
| 开户时间 | 即时 | 1-3 个工作日 | 即时 | 即时 |

---

## 费率对比

| | Stripe | Airwallex | Paddle | LemonSqueezy |
|---|---|---|---|---|
| 信用卡 | 2.9% + $0.30 | 3.3% + $0.30 | 5% + $0.50 | 5% + $0.50 |
| 支付宝/微信 | $0 | 2.9% | 不支持 | 不支持 |
| 提现 | 按汇率 | 0 费用 | 包含在费率里 | 包含在费率里 |
| 外汇加价 | 1% | 0.5% | $0 | $0 |

**举例：** 一笔 $50 的订单
- Stripe：$50 - $1.75 = **到手 $48.25**
- Airwallex：$50 - $1.95 = **到手 $48.05**
- Paddle（含税代缴）：$50 - $3.00 = **到手 $47.00**
- LemonSqueezy（含税代缴）：$50 - $3.00 = **到手 $47.00**

费率差异不大。真正值钱的是 Paddle 和 LemonSqueezy 帮你搞定全球税务（欧盟 VAT、美国 Sales Tax）。

---

## 国内开发者最优解

### 方案 A：最低门槛
LemonSqueezy → 中国主体可注册 → 自动处理全球税务 → 每周提现

### 方案 B：最专业
注册美国公司（Stripe Atlas 或 Firstbase）→ 用 Stripe → 费率最低 → API 最强

### 方案 C：国内主体 + 对公
空中云汇 → 支持中国大陆企业 → 提现到国内银行 → 有支付宝/微信支付

---

## 结论

**刚起步 → LemonSqueezy 或 Paddle，不需要注册公司就能收钱。**
