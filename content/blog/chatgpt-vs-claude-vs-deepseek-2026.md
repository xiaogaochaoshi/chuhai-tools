## 各模型特点

### Claude（综合最强）

**编程总分第一**。系统设计和 Bug 修复特别强，1M token 上下文让你可以把整个代码库扔进去。Claude Code CLI 是目前最好的终端 AI 编程体验。

**适合：** 复杂项目重构、大型代码库理解、系统架构设计

**价格：** Pro $20/月，Max $100-200/月

### ChatGPT（文档和 SQL 最强）

GPT-5 在 API 文档生成和 SQL 方面表现出色。生态最完善，GPTs 商店有大量编程专用 GPT。

**适合：** 全栈快速开发、需要丰富生态

**价格：** Plus $20/月，Pro $200/月

### DeepSeek（性价比之王）

编程总分比 GPT-5 低 7 分，但**价格是 1/20**。算法和数学推理得分很高（R1 模型功劳）。中文编程体验不输 Claude。

**适合：** 预算敏感、中文项目、大批量 API 调用

**价格：** 网页版免费，API ¥1/百万 token

## 真实使用建议

**日常开发：** Claude Code + DeepSeek API（Claude 处理复杂任务，DeepSeek 批量调用省钱）

**预算充足：** Claude 订阅 + Cursor（Claude 模型集成）

**零成本方案：** DeepSeek 网页版 + Trae IDE（两个都免费，效果不差）
