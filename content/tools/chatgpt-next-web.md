## 简介

ChatGPT-Next-Web 是最流行的 ChatGPT 开源前端之一，GitHub 80k+ Star。支持一键部署到 Vercel，无需服务器、无需写代码。

## 核心功能

- 支持 GPT-4、GPT-4o、Claude 等模型
- 对话记录本地存储，隐私安全
- Markdown 渲染、代码高亮、LaTeX 公式
- 支持自定义域名和密码保护
- PWA 支持，可安装到手机桌面

## 一键部署

点击上方"一键部署"按钮，登录 Vercel 后填入你的 OpenAI API Key 即可。3 分钟上线，零成本。
