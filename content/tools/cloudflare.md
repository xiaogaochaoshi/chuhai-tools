## 简介

Cloudflare 是全球最大的网络基础设施平台，但它的价值远不止 CDN。Cloudflare Workers（边缘函数）、D1（SQLite 数据库）、R2（对象存储）构成了一个完整的全栈部署方案。

## 为什么推荐

- **全球边缘网络**：330+ 个数据中心，延迟极低
- **Workers**：边缘函数，冷启动几乎为零
- **D1 + R2**：轻量数据库 + S3 兼容存储
- **免费版大方**：CDN、DNS、DDoS 防护全免费

## 定价

- 免费版：CDN + DNS + DDoS 防护 + Workers 10万次/天
- Pro：$20/月
- Workers Paid：$5/10M 请求

## 适合谁

需要极致性能的独立开发者。Cloudflare Workers + D1 适合轻量 API 和边缘渲染场景。
