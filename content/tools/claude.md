## 简介

Claude 由 Anthropic 开发，是目前编程能力最强的大模型之一。Claude Opus 4.7 支持 1M token 上下文，Claude Code CLI 工具在开发者圈迅速走红。

## 核心能力

- Opus 4.7 / Sonnet 4.6 / Haiku 4.5 三档模型
- 1M token 超长上下文（可一次读完整个代码库）
- 编程能力顶尖，复杂重构和系统设计表现出色
- Artifacts：实时预览代码/文档/图表
- 安全对齐，输出风格偏谨慎和 helpful
- Claude Code：终端里的 AI 编程助手

## 定价

- 免费版：Sonnet 模型，有限次数
- Pro：$20/月，全模型，无限使用
- Max：$100/月（高优先级）或 $200/月（最高优先级）
