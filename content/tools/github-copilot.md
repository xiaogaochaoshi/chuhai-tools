## 简介

GitHub Copilot 是 AI 编程助手的开创者，由 OpenAI Codex 驱动。与 Cursor 强调"全库理解"不同，Copilot 的优势在于补全速度快、IDE 集成最深。

## 为什么推荐

- **补全速度极快**：毫秒级延迟，不打断心流
- **GitHub 生态**：与 Pull Request、Issues 深度联动
- **Agent 模式**：Copilot Agent 能自主完成复杂任务

## 定价

- 个人版：$10/月（年付 $100）
- 商业版：$19/月/人
- 企业版：$39/月/人

## vs Cursor 怎么选

如果你需要整库重构和上下文理解，选 Cursor。如果追求极快的补全速度和 GitHub 原生体验，选 Copilot。两者可以共存。
