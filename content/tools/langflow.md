## 简介

LangFlow 是 LangChain 生态的可视化工作流编辑器，GitHub 45k+ Star。用拖拽方式组合 LLM、Memory、Tool、Chain 等组件。

## 核心功能

- 完整的 LangChain 组件库可视化
- 实时预览和调试
- 导出为 Python 代码或 JSON
- 支持自定义组件
- API 端点一键生成

## 一键部署

通过 Railway 或 HuggingFace Spaces 一键部署。无需配置服务器，即开即用。
