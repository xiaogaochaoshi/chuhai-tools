## 简介

Claude Code 是 Anthropic 推出的命令行 AI 编程助手，直接在终端里运行。它不只是代码补全——它能读代码库、执行命令、修改文件、提交代码，像一个真正的全栈工程师一样工作。

## 为什么推荐

- **终端原生**：不需要打开 IDE，SSH 到服务器也能用
- **上下文超大**：Claude 的上下文窗口能装下整个代码库
- **自主执行**：能自己运行测试、修 bug、迭代直到通过
- **Git 集成**：自动提交、创建 PR，工作流无缝

## 定价

- Max 计划：$100/月（推荐）
- 按量计费：API 价格

## 适合谁

喜欢命令行的开发者。特别适合需要频繁在服务器上改代码、做 DevOps 的独立开发者。
