## 简介

Cloudflare AI Gateway 让你的应用通过一个统一端点调用所有 AI 服务，自动获得缓存、速率限制、用量分析和成本控制能力。

## 核心功能

- 统一代理 OpenAI / Claude / Groq / Replicate 等所有 AI API
- 响应缓存（相同请求不重复计费）
- 速率限制和用量控制
- 完整的日志和分析仪表盘
- 零配置，改一行代码即可接入
- Cloudflare 全球网络低延迟

## 一键部署

登录 Cloudflare 后台 → AI Gateway → 创建网关 → 替换 API Base URL 即可。完全免费（含 10 万次请求/月）。
