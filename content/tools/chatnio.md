## 简介

Chat Nio 是一个功能全面的 AI 对话商业平台，GitHub 5k+ Star。不止是聊天前端，还内置了用户系统、计费、分销等商业化功能。

## 核心功能

- 多模型聚合（50+ 模型）
- 联网搜索 + 文件解析
- 用户管理 + 计费系统 + 分销返佣
- 管理后台 + 数据统计
- Markdown / Mermaid 图表 / 代码渲染
- 企业级安全与隐私
