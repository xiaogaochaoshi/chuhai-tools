## 简介

Dify 是国内最知名的开源 LLMOps 平台，GitHub 70k+ Star。提供可视化编排界面，让非开发者也能搭建 AI 应用。

## 核心功能

- 可视化 AI 工作流编排（拖拽式）
- RAG 知识库引擎，支持多种文档格式
- Agent 智能体，可调用工具和 API
- 对话日志与分析仪表盘
- 多模型支持，灵活切换
- API 发布，嵌入自己的产品

## 一键部署

[Dify Cloud](https://cloud.dify.ai) 提供免费托管版本。自部署可使用 Docker Compose 或 Railway 一键部署。
