## 简介

Cursor 是目前最受欢迎的 AI 编程工具之一，基于 VS Code 深度定制。与 GitHub Copilot 不同，Cursor 能理解你的整个代码库上下文，支持 Tab 补全、行内编辑和聊天式交互。

## 为什么推荐

- **整库上下文**：不像 Copilot 只看到当前文件，Cursor 能索引整个项目
- **Composer 模式**：一次性修改多个文件，重构效率极高
- **迁移成本低**：从 VS Code 迁移只需导入配置，插件兼容

## 定价

- 免费版：每月 2000 次补全 + 50 次高级请求
- Pro：$20/月，无限补全 + 500 次高级请求
- Business：$40/月/人

## 适合谁

所有写代码的人。尤其适合全栈独立开发者，一个工具覆盖前后端。
