## 简介

Windsurf 是 Codeium 推出的 AI IDE，最大的特色是 Cascade 模式——AI 像瀑布一样自动推理和编辑多个文件，不需要手动指定范围。

## 为什么推荐

- **Cascade 自动推理**：写好 prompt，自动分析影响范围并修改
- **免费额度大**：免费版几乎够个人开发者日常使用
- **独立产品**：不绑 VS Code，有自己的 UI 设计理念

## 定价

- 免费版：无限补全 + 每月 200 Cascade 步骤
- Pro：$15/月
- 企业版：$35/月/人

## 适合谁

想尝试非 VS Code 系 AI IDE 的开发者。Cascade 模式的自动化程度比 Cursor Composer 更高。
