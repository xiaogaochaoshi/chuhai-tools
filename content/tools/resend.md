## 简介

Resend 是专为开发者设计的事务邮件服务。最大的亮点是能用 React 组件写邮件模板，所见即所得。发送速度极快，API 设计简洁。

## 为什么推荐

- **React 邮件模板**：用 React 写邮件，实时预览，不用再手写 HTML 邮件
- **发送速度快**：平均 400ms 送达
- **免费额度大**：每天 100 封免费
- **广播模式**：支持 Newsletter 群发，不只是事务邮件

## 定价

- 免费版：100 封/天
- Starter：$20/月（5,000 封/月）
- Growth：$50/月（50,000 封/月）

所有需要发送邮件的独立开发者。Resend 的 React 邮件模板是杀手级体验。
