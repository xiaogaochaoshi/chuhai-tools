## 简介

FastGPT 是国内开发者最常用的开源知识库问答平台，GitHub 20k+ Star。专注于将文档转化为 AI 知识库，支持复杂的数据预处理。

## 核心功能

- 文档自动分段、向量化、QA 拆分
- 工作流编排（可视化）
- API 开放接口和 Webhook
- 多模型支持：OpenAI / 文心 / 千问 / DeepSeek 等
- 多租户管理
- 对话日志与统计分析

## 一键部署

使用 [Sealos](https://sealos.run) 或 Docker Compose 一键部署。官方提供免费云版本可体验。
