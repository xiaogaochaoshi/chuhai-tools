## 简介

Open WebUI（原 Ollama WebUI）是功能最全面的自托管 AI 前端，GitHub 60k+ Star。从 Ollama 起步，现已支持所有 OpenAI 兼容 API。

## 核心功能

- 类 ChatGPT 的完整 UI 体验
- 支持 Ollama 本地模型 + OpenAI 兼容 API
- RAG 文档问答（PDF、Word、TXT 等）
- 多用户管理 + RBAC 权限控制
- Web 搜索集成
- TTS / STT 语音交互
- 插件系统可扩展

## 一键部署

支持 Docker 一键启动。推荐用 Railway 或 Sealos 云端部署。
