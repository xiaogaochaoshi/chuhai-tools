## 简介

Trae 是字节跳动推出的 AI 编程 IDE，基于 VS Code 深度定制。内置 Claude 4 和 GPT-5 模型，完全免费使用，对中文开发者体验极佳。

## 核心功能

- 内置 Claude 4 / GPT-5，免费无限使用
- Builder 模式：一句话描述需求，自动生成完整项目
- Chat 模式：边写代码边对话
- 中文理解能力强，支持中文注释生成
- 支持图片上传，根据截图生成代码
- 一键导入 VS Code 配置和插件

## 适用场景

独立开发者、全栈项目、原型验证。目前只有 macOS 版本，Windows 版即将推出。
