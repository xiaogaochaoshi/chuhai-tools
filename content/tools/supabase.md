## 简介

Supabase 是 Firebase 的开源替代方案，底层是 PostgreSQL。它提供数据库、认证、存储、Edge Functions 四大核心模块，是独立开发者的全栈后端首选。

## 为什么推荐

- **PostgreSQL 内核**：真正的 SQL 数据库，不是 NoSQL 黑盒
- **Auth 开箱即用**：支持邮箱、OAuth、手机验证码登录
- **实时订阅**：数据库变更自动推送到前端
- **免费额度大**：免费版包含 500MB 数据库 + 50,000 月活用户

## 定价

- 免费版：500MB 数据库，50K MAU
- Pro：$25/月/项目
- Team：$599/月

## 适合谁

每个独立开发者。Supabase + Next.js 是当前出海产品的黄金组合。
