## 简介

Flowise 是基于 LangChain 的低代码 AI 应用构建平台，GitHub 35k+ Star。节点拖拽即可搭建复杂的 LLM 应用。

## 核心功能

- 可视化节点编辑器（基于 LangChain）
- 支持自定义工具和 Agent
- 对话嵌入 iframe，集成到任何网站
- 向量数据库集成（Pinecone、Weaviate 等）
- API 端点自动生成
